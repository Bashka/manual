====== Описание ======

Pathauto - это модуль, автоматизирующий процесс формирования псевдонимов URL-путей по настроенным шаблонам.

====== Использование ======

Интерфейс модуля расположен на экране "Синонимы URL" в конфигурации системы. Вкладка "Шаблоны" позволяет указать шаблон URL-пути для конкретного элемента системы (типа материала, таксономии и т.д.).

После конфигурирования шаблонов URL-путей необходимо сгенерировать псевдонимы для всех существующих элементов системы. Для этого используется вкладка "Массовое обновление".
