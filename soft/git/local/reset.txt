====== Сброс индекса ======

Если необходимо исключить конкретный файл из индекса, используется команда ''%%git reset -- адрес%%''. Команда ''git reset'' отменяет весь индекс.

====== Сброс комита ======

Команда ''git reset'' способна отменять комиты. Делается это двумя способами:

  - Мягкий (''%%--soft%%'') - сброс без удаления комита
  - Жесткий (''%%--hard%%'') - сброс с удалением комита

===== Мягкий =====

Мягкий сброс выполняется с помощью команды ''%%git reset --soft комит%%''. Он сбрасывает все комиты до указанного, сохраняя индекс и сами комиты так, чтобы к ним можно было вернуться в будущем.

Пример:

  # Отмена последнего комита с добавлением его содержимого в индекс
  git reset --soft HEAD^
  # Создание нового файла с добавлением его в индекс
  touch new.txt && git add new.txt
  # Перезапись отмененного ранее комита (ORIG_HEAD указывает на отмененный комит)
  git commit -c ORIG_HEAD

===== Жесткий =====

Жесткий сброс выполняется с помощью команды ''%%git reset --hard комит%%''. Он удаляет все комиты до указанного. Данная команда может быть использована для удаления части истории комитов.

Пример:

  # Удаление последнего комита
  git reset --hard HEAD^
