====== Список веток ======

Ветка представляет указатель на некоторый комит, от которого будет отсчитываться новая история комитов (ветка комитов).

Для просмотра списка всех доступных веток, используется команда ''%%git branch --list%%''.

Пример:

  $>git branch --list
  * master
    new

Символ * маркерует текущую ветку.

Команда имеет следующие ключи:

  * ''%%git branch --list --all%%'' - дополнить список ветками Git серверов
<code>
$>git branch --list --all
* master
  remotes/origin/HEAD -> origin/master
  remotes/origin/master
</code>
  * ''%%git branch --list -r%%'' - показать только ветки Git серверов
  * ''%%git branch --list --contains комит%%'' - показать ветки, содержащие заданный комит
  * ''%%git branch --list --verbose%%'' - дополнить список веток подробной информацией о них (хэш-сумма последнего комита и его описание)
  * ''%%git branch --list --merged%%'' - показать только ветки, слитые (merge) с текущей
  * ''%%git branch --list --no-merged%%'' - показать только ветки, не слитые (merge) с текущей

====== Создание ветки ======

Для создания новой ветки необходимо:

  - Перейти на корневой комит новой ветки с помощью команды ''git checkout комит''
  - Создать ветку с помощью команды ''git branch имя''
  - Перейти на созданную ветку с помощью команды ''git checkout ветка''

Пример:

  git checkout 8d52
  git branch newBranch
  git checkout newBranch

Для двух последних команд из примера существует следующее сокращение - ''%%git checkout -b имя%%''. Эта команда создает новую ветку и делает ее текущей.

====== Переход между ветками ======

Чтобы сделать некоторую ветку текущей (для нового репозитория текущей является ветка ''master'') используется команда ''git checkout ветка''.

====== Переименование ветки ======

Если необходимо сменить имя ветки, используется команда ''%%git branch -m имя новоеИмя%%''.

====== Удаление ветки ======

Для удаления ветки используется команда ''%%git branch -d ветка%%'', но ветка не будет удалена, если ее изменения небыли слиты (merge) с другой веткой. Для принудительного удаления ветки используется команда ''%%git branch -D ветка%%''.
