====== Создание репозитория ======

Для создания нового, локального репозитория в текущем каталоге, необходимо выполнить команду ''%%git init%%'', которая создаст каталог ''.git'' в текущем каталоге. Данный каталог и является репозиторием Git.

Пример:

  cd ~/myProj
  git init

После создания локального репозитория он пуст. Для добавления файлов под версионный контроль, необходимо выполнить коммит.

Команда имеет следующие ключи:

  * ''%%git init -q ...%%'' - "тихий" режим. Не выводится ничего, кроме ошибок и предупреждений
  * ''%%git init --template=каталог ...%%'' - создание репозитория на основе шаблона. В созданный репозиторий будут скопированы все файлы шаблонного репозитория
