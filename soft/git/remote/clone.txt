====== Клонирование из сервера ======

Для работы с Git-сервером необходимо создать его копию на локальной машине. Для этого используется команда ''git clone url'', которая создает точную копию внешнего репозитория, а так же:

  * Создает псевдоним ''origin'', который ссылается на URL адрес сервера
  * Создает ветку ''master'', которая отслеживает ветку ''origin/master''

Если необходимо загрузить внешний репозиторий в некоторый каталог, используется команда ''git clone url каталог''.

Команда имеет следующие ключи:

  * ''%%git clone -q ...%%'' - "тихий" режим. Не выводится ничего, кроме ошибок и предупреждений
  * ''%%git clone --template=каталог ...%%'' - создание репозитория на основе шаблона. В созданный репозиторий будут скопированы все файлы шаблонного репозитория
  * ''%%git clone -l адрес каталог ...%%'' - клонирование локального репозитория

  git clone -l ~/myProj ~/newProj

  * ''%%git clone --progress ...%%'' - демонстрация прогресс-бара
  * ''%%git clone --branch ветка ...%%'' - задать текущую (''master'') ветку после клонирования (по умолчанию используется ветка, на которую указывает HEAD)
  * ''%%git clone --origin имя ...%%'' - задать псевдоним репозитория, из которого выполняется клонирование (по умолчанию используется имя ''origin'')
