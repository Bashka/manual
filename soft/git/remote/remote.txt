====== Добавление Git-сервера ======

Для взаимодействия с Git-сервером необходимо определить его псевдоним в локальном репозитории. Делается это с помощью команды ''git remote add псевдоним url''.

Пример:

  git remote add teamA https://site.com/project.git

Команда имеет следующие ключи:

  * ''%%git remote add --tags ...%%'' - импортировать все маркеры (tags) из внешнего репозитория

====== Список псевдонимов ======

Для просмотра списка всех добавленных Git-серверов, используется команда ''git remote''.

Команда имеет следующие ключи:

  * ''%%git remote -v%%'' - указание URL адреса, на который ссылается каждый псевдоним
<code>
  $>git remote
  origin	site.com:/home/user/project.git (fetch)
  origin	site.com:/home/user/project.git (push)
</code>

====== Удаление псевдонима ======

Для удаления псевдонима Git-сервера используется команда ''git remote rm псевдоним''.

====== Переименование ======

Для переименования псевдонима Git-сервера используется команда ''git remote rename псевдоним новыйПсевдоним''.
