====== Отслеживаемые ветки ======

Команды ''git push'' и ''git pull'' без параметров работают с отслеживаемыми ветками. Отслеживаемыми являются те локальные ветки, для которых определен псевдоним сервера и внешняя ветка по умолчанию. Для отслеживаемых веток Git "знает" куда/откуда следует выгружать/загружать изменения, что не требует указания дополнительных параметров.

====== Создание отслеживаемой ветки ======

Если необходимо создать отслеживаемую ветку, используется два подхода:

  * ''%%git branch --set-upstream ветка псевдоним/внешняяВетка%%'' - данная команда делает локальную ветку отслеживаемой по отношению к некоторой внешней ветке
  * ''%%git push --set-upstream псевдоним ветка%%'' - данная команда выгружает изменения из локальной ветки на сервер и делает ее отслеживаемой по отношению к ветке ''псевдоним/ветка''
  * ''%%git push псевдоним ветка:внешняяВетка%%'' - данная команда выгружает изменения из локальной ветки на сервер и делает ее отслеживаемой по отношению к данной внешней ветке
