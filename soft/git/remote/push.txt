====== Выгрузка текущей ветки ======

Для выгрузки состояния текущей ветки на сервер используется команда ''git push''.

Данная команда работает с отслеживаемыми ветками и выгружает все изменения, которые отсутствуют на сервере. Если же необходимо выгрузить не отслеживаемую ветку, используется команда ''git push псевдоним обычнаяВетка:внешняяВетка''.

Для выгрузки на некоторый сервер, используется команда ''git push псевдоним''.

====== Удаление ветки на сервере ======

Если необходимо удалить внешнюю ветку с сервера используется команда ''git push псевдоним :внешняяВетка''.
