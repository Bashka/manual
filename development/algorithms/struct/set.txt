====== Описание ======
Множество - это неупорядоченная совокупность элементов без повторов (аналог математического множества). Множества применяются только для проверки вхождения некоторого значения или другого множества в данное множество.

====== Реализация ======
===== На логическом массиве =====
Для реализации необходимо:
  * Определить массив логических элементов.

==== Операции ====
Проверка вхождения элемента:
<code>
return array[index]
</code>
где:
  * ''array'' - логический массив множества;
  * ''index'' - идентификатор проверяемого элемента.

Добавление элемента:
<code>
array[index] = true
</code>
где:
  * ''array'' - логический массив множества;
  * ''index'' - идентификатор добавляемого элемента.

Удаление элемента:
<code>
array[index] = false
</code>
где:
  * ''array'' - логический массив множества;
  * ''index'' - идентификатор удаляемого элемента.

===== На на битовой маске =====
Для реализации необходимо:
  * Определить переменную необходимой длины.

==== Операции ====
Проверка вхождения элемента (примерные значения):
<code>
101100 & 000100 > 0
</code>

Добавление элемента (примерные значения):
<code>
101100 | 000001 -> 101101
</code>

Удаление элемента (примерные значения):
<code>
101101 & 111110 -> 101100
</code>

====== Свойства ======
^ Свойство ^ Значение ^
|Повторение элементов|Нет|
|Упорядоченность элементов|Нет|
|Доступ к элементу|Проверка наличия|

===== Положительные =====
  * Изменение элемента
  * Проверка присутствия

===== Отрицательные =====
  * Именение размеров
  * Хранение только логических данных
