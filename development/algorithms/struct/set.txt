====== Описание ======
Множество - это неупорядоченная совокупность элементов без повторов (аналог математического множества). Множества применяются только для проверки вхождения некоторого значения или другого множества в данное множество.

====== Свойства ======
^ Свойство ^ Значение ^
|Повторение элементов|Нет|
|Упорядоченность элементов|Нет|
|Доступ к элементу|Проверка наличия|

====== Реализации ======
===== На логическом массиве =====
  * Элемент коллекции это данные логического типа;
  * Все элементы коллекции расположены в массиве;
  * Индекс элемента логически связан со значением множества.

===== Плюсы =====
  * Проверка присутствия
  * Изменение структуры

===== Минусы =====
  * Потребление памяти

==== Операции ====
<code>
class SetVector
  Vector store
  
  function Set(size)
    store = new Vector(bool, size)
    for i = 0; i < size; i++
      remove(i)
  
  function is(index)
    return store[index]
    
  function add(index)
    store[index] = true

  function remove(index)
    store[index] = false
</code>

===== На битовой маске =====
  * Элемент коллекции это бит данных
  * Все элементы коллекции расположены в достаточном числа байт;
  * Разрядность бита логически связана со значением множества.

===== Плюсы =====
  * Проверка присутствия
  * Потребление памяти

===== Минусы =====
  * Именение структуры
  * Хранение только логических данных

==== Операции ====
<code>
class SetMask
  int mask
  
  function Set()
    mask = 0
  
  function getMask(index)
    if index == 0
      indexMask = 0
    else
      indexMask = 1
      while index--
        indexMask = indexMask << 1
    return indexMask
  
  function is(index)
    return mask & getMask(index) > 0
    
  function add(index)
    mask |= getMask(index)

  function remove(index)
    mask &= ~getMask(index)
</code>
