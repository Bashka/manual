====== Описание ======
Массив - это совокупность элементов одного типа, расположенных последовательно в памяти.

====== Реализация ======
Для реализации необходимо:
  * Определить блок памяти длиной, достаточной для хранения данных заданного типа;
  * Выделить необходимое число блоков (равное длине массива) в памяти;
  * Получить ссылку на первый блок.

===== Операции =====
Получения элемента:
<code>
return afblock + index * length
</code>
где:
  * ''afblock'' - адрес памяти первого элемента;
  * ''index'' - индекс целевого элемента;
  * ''length'' - длина блока.

Увеличения длины:
<code>
newarray[newsize]
for index in oldarray
  newarray[index] = oldarray[index]
delete oldarray
</code>
где:
  * ''newsize'' - целевая длина массива;
  * ''oldarray'' - текущий массив;

Уменьшения длины:
<code>
newarray[newsize]
for index in oldarray
  if index > newsize
    break
  newarray[index] = oldarray[index]
delete oldarray
</code>
где:
  * ''newsize'' - целевая длина массива;
  * ''oldarray'' - текущий массив;

Поиск:
<code>
for index in array
  if array[index] == target
    return index
</code>
где:
  * ''target'' - искомое значение;
  * ''array'' - исходный массив.

====== Свойства ======
^ Свойство ^ Значение ^
|Повторение элементов|Да|
|Упорядоченность элементов|Да|
|Доступ к элементу|По индексу (порядковому номеру)|

===== Положительные =====
  * Доступ к элементу
  * Изменение элемента

===== Отрицательные =====
  * Изменение размеров
  * Поиск
