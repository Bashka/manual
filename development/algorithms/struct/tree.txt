====== Описание ======
Дерево - это иерархическая структура, состоящая из совокупности элементов, связанных отношением родитель-потомок. Элемент, не имеющий родителя называется корневым, а не имеющий потомков - листовым.

====== Свойства ======
^ Свойство ^ Значение ^
|Повторение элементов|Да|
|Упорядоченность элементов|Да|
|Доступ к элементу|Перебором|

====== Реализация ======
===== Восходящие связи =====
{{ .:media:tree_up.png |Схема связей}}
Для реализации необходимо:
  * Определить блок памяти длиной, достаточной для хранения данныхзаданного типа и указателя на такой блок;
  * Каждый блок (кроме корня дерева) должен ссылаться на родительский блок;
  * Получить ссылки на листовые блоки дерева.

==== Плюсы ====
  * Восходящий перебор
  * Изменение структуры

==== Минусы ====
  * Поиск
  * Нисходящий перебор

===== На бинарных списках =====
{{ .:media:tree_down.png |Схема связей}}
Для реализации необходимо:
  * Определить блок памяти длиной, достаточной для хранения данныхзаданного типа и два указателя на такой блок;
  * Первый указатель блока служит для хранения ссылки на список дочерних элементов;
  * Второй указатель блока служит для хранения ссылки на следующий элемент списка того же уровня иерархии;
  * Получить ссылку на корневой элемент.

==== Плюсы ====
  * Нисходящий перебор
  * Изменение структуры
