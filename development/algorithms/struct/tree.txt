====== Описание ======
Дерево - это иерархическая структура, состоящая из совокупности элементов, связанных отношением родитель-потомок. Элемент, не имеющий родителя называется корневым, а не имеющий потомков - листовым.

====== Свойства ======
^ Свойство ^ Значение ^
|Повторение элементов|Да|
|Упорядоченность элементов|Да|
|Доступ к элементу|Перебором|

====== Реализации ======
===== Восходящие связи =====
{{ .:media:tree_up.png |Схема связей}}
  * Элемент коллекции это структура, хранящая данные и указатель на такой элемент;
  * Каждый элемент (кроме корня) должен ссылаться на родительский элемент;
  * Структура хранит ссылки на листовые элементы.

==== Плюсы ====
  * Восходящий перебор
  * Изменение структуры

==== Минусы ====
  * Поиск
  * Нисходящий перебор

==== Операции ====
<code>
class TreeUp
  class Item
    mixed value
    Item* parent
  
  List leaves
</code>

===== На бинарных списках =====
{{ .:media:tree_down.png |Схема связей}}
  * Элемент коллекции это структура, хранящая данные и два указателя на такие же элементы;
  * Каждый элемент хранит следующие указатели: на братский элемент и на первый дочерний элемент;
  * Структура хранит указатель на корневой элемент.

==== Плюсы ====
  * Нисходящий перебор
  * Изменение структуры

==== Минусы ====
  * Поиск
  * Восходящий перебор

==== Операции ====
<code>
class TreeList
  class Item
    mixed value
    Item* brother
    Item* firstChild
  
  Item* root
</code>
