====== Описание ======
Список - это совокупность элементов одного типа, распределенных в памяти. Каждый элемент списка помимо значения, хранит ссылку на следующий элемент того же списка.

====== Реализация ======
Для реализации необходимо:
  * Определить блок памяти длиной, достаточной для хранения данных заданного типа и указателя на такой блок;
  * Каждый блок (кроме последнего) должен ссылаться на соседний блок, связывая элементы списка;
  * Получить ссылку на первый блок.

===== Операции =====
Получение элемента:
<code>
ablock = afblock
while index--
  ablock = ablock.next
return ablock
</code>
где:
  * ''afblock'' - адрес памяти первого элемента;
  * ''index'' - индекс целевого элемента.

Увеличения длины:
<code>
ablock = list[index]
ablock.next = newblock
</code>
где:
  * ''list'' - исходный список;
  * ''index'' - индекс элемента, после которого вставляется новый элемент.

Уменьшения длины:
<code>
pablock = list[index - 1]
ablock = pablock.next
pablock.next = null
delete ablock
</code>
где:
  * ''list'' - исходный список;
  * ''index'' - индекс удаляемого элемента.

Поиск:
<code>
ablock = afblock
do
  if ablock.val == target
    return ablock
while ablock = ablock.next
</code>
где:
  * ''target'' - искомое значение;
  * ''afblock'' - адрес памяти первого элемента.

====== Свойства ======
^ Свойство ^ Значение ^
|Повторение элементов|Да|
|Упорядоченность элементов|Да|
|Доступ к элементу|Перебором|

===== Положительные =====
  * Именение размеров
  * Изменение элемента

===== Отрицательные =====
  * Доступ к элементу
  * Поиск
