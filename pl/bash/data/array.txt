====== Объявление и использование ======
Массивы представляют собой плоский список слов. Для инициализации массива используется запись вида:
<code bash>
переменная=(словоA словоB ...)
</code>
Здесь каждое слово является отдельным элементом массива (индексация начинается с нуля).

Так же можно использовать для инициализации массива результат работы команды:
<code bash>
arr=( `ps` )
</code>

Команда ''eval'' позволяет создавать вложенные массивы:
<code bash>
arr=( "( 1 2 )" )
eval inarr=${arr[0]}
echo ${inarr[1]} # 2
</code>

Для доступа к элементу массива используется запись вида:
<code bash>
${переменная[индекс]}
</code>
При этом запись заменяется значением элемента под данным индексом. Если элемент не определен, используется пустая строка.

Конкретный элемент массива можно установить следующим способом:
<code bash>
переменная[индекс]=значение
</code>

Ниже приведен пример объявления и использования простого массива дней недели:
<code bash>
week=('Пн' 'Вт' 'Ср' 'Чт' 'Пт' 'Сб' 'Вс');
week[0]='Вс';
week[6]='Пн';
echo ${week[0]};
</code>

====== Операции ======
  * Получение элементов массива в виде отдельных слов;
<code bash>
arr=('Hello world' 'Goodbye world')
elem=${arr[@]} # Аналоги elem='Hello world' 'Goodbye world'
</code>
  * Определение числа элементов в массиве;
<code bash>
${#переменная[@]}
</code>
  * Удаление элементов массива;
<code bash>
unset имя[индекс]
</code>
  * Копирование массива в новый массив;
<code bash>
arr=(...)
arr2=("${arr[@]}" ...)
</code>
  * Добавление нового элемента в конец массива.
<code bash>
arr[${#arr[@]}]=значение
</code>
