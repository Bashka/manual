====== Структура ======
Структура ''case'' является реализацией ''switch'' в Bash. Она имеет вид:
<code bash>
case строка in
  шаблон)
    блок
    ;;
  *)
    блокПоУмолчанию
esac
</code>
Исполяемый блок выбирается при соответствии шаблона исходной строке. Можно указать несколько шаблонов, разделенных символом ''|''.

Важно помнить, что удаление символа '';;'' в блоке недопустимо, при этом после выполнения одного блока, управление сразу переходит за структуру ''case''.
<code bash>
case 'test' in
  test )
    echo 1 # Блок выполнится
    ;;
  te*t )
    echo 2 # Блок не выполнится
    ;;
esac
</code>

Пример определения нажатия клавишь со стрелками:
<code bash>
while test 1 -eq 1; do
  read -s -n1 chr

  case $chr in
    \A)
      echo 'Up'
      ;;
    \B)
      echo 'Down'
      ;;
  esac
done
</code>
