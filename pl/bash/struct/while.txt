====== Структура ======
Цикл ''while'' в Bash позволяет выполнять блок кода до тех пор, пока команда условия возвращает ноль. Его структура:
<code bash>
while командаУсловия; do
  тело
done
</code>

Цикл ''until'' аналогичен ''while'' с той лишь разницей, что выполняется до тех пор, пока команда условия возвращает отличное от нуля значение. Его структура:
<code bash>
until командаУсловия; do
  тело
done
</code>

Пример цикла ''while'':
<code bash>
i=0
while test $i -lt 10; do # Аналог while [ test $i -lt 10 ]; do
  echo $i
  (( i++ ))
done
</code>

Использование C-подобного синтаксиса:
<code bash>
i=0
while (( $i < 10 )); do
  echo $i
  (( i++ ))
done
</code>

Пример цикла ''while'' для прохода по строкам файла:
<code bash>
while read line; do # Построчное чтение потока ввода
  echo "$line"
done < test.txt # Поток ввода получен из файла
</code>
