====== Структура ======
Структура ветвления ''if'' на языке Bash имеет вид:
<code bash>
if команда; then
  блокИстина
elif команда; then
  блокИстина
else
  блокЛожь
fi
</code>
Ветви ''elif'' и ''else'' являются необязательными.

Структура ожидает возвращаемое командой значение и выполняет блок "Истина" только в том случае, если это значение равно нулю, иначе выполняется следующая проверка или блок ''else''.

Важно помнить, что проверяется именно возвращаемое командой значение, а не данные, передаваемые в стандартный поток вывода.

Структура ожидает именно команду, потому нельзя использовать запись вида:
<code bash>
if $var; then # Ошибка
  ...
fi
</code>
Так как значение переменной будет интерпретированно структурой как команда.

Подходящей для структуры командой является утилита [[..:data:boolean#Операторы|test]], которая возвращает ноль в случае истинности условия, и отличное от нуля значение в случае его ложности. У данной утилиты есть псевдоним ''[ ... ]'':
<code bash>
if [ $a -eq $b ]; then # Аналог: test $a -ea $b
  ...
fi
</code>
