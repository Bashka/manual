====== Структура ======
Цикл ''for'' в Bash предназначен для поэлементного прохода по "словам" в строке или элементам массива. Его структура:
<code bash>
for переменная in массив; do
  тело
done
</code>
Здесь в качестве массива может выступать строка, словами в которой будут выступать данные, разделенные пробелом.
<code bash>
for file in `ls`; do
  echo $file
done
</code>

Если перебираемая в цикле строка возвращается сторонней утилитой, то можно выбрать символ, по которому она будет разделена на слова. Для этого используется переменная ''IFS'':
<code bash>
IFS="\n"
for process in `ps`; do # По одному процессу за итерацию
  echo $process
done
</code>

Пример прохода по файлам каталога:
<code bash>
for file in ./*; do
  echo $file
done
</code>

Пример прохода по ключам массива:
<code bash>
arr=('a' 'b' 'c')
for i in $(seq ${#arr[@]}); do
  i=`expr $i - 1`
  echo $i = ${arr[$i]}
done
</code>

Есть более C-подобный синтаксис цикла ''for'':
<code bash>
for (( инициализация; условие; шаг )); do
  тело
done
</code>

Пример C-подобной структуры ''for'':
<code bash>
for (( i = 10, j = 0; $i >= 1; i--, j++ )); do
  echo $i
done
</code>
