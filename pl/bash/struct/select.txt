====== Структура ======
Структура ''select'' позволяет сформировать меню выбора. Структура имеет вид:
<code bash>
select переменная in массив; do
  ...
done
</code>
Здесь в качестве массива может выступать строка, словами в которой будут выступать данные, разделенные пробелом.

Выбранный пользователем пункт меню будет записан в переменную, после чего пользователю будет повторно предложено выбрать один из пунктов меню. Для выхода из цикла используется команда ''break''.

Пример:
<code bash>
select file in $( ls ) 'quit'; do
  if [[ $file == 'quit' ]]; then
    break
  else
    echo 'Select file: '$file
  fi
done
</code>
