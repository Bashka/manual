<code bash>
read переменная ...
</code>
Утилита считывает в переменную данные из потока (по умолчанию используется поток ввода сценария) до достижения символа новой строки. Если указано несколько переменных, данные будут записаны в них пословно.

Параметры:
  * ''-n число'' - число считываемых байт. После получения указанного количества байт утилита прекратит считывание данных автоматически. При использовании данного параметра символ новой строки не приводит к завершению считывания данных из потока;
  * ''-p сообщение'' - вывод сообщения (в поток вывода команды), приглашающего пользователя ввести данные;
  * ''-t сек'' - тайм-аут чтения данных из потока (в секундах);
  * ''-s'' - чтение данных без повтора их в потоке вывода команды.

Пример чтения из стандартного потока ввода:
<code bash>
read -p 'Введите имя и фамилию: ' name surname
echo 'Ваше имя: '$name
echo 'Ваша фамилия: '$surname
</code>
Пример чтения из файла:
<code bash>
while read line; do
  echo "$line"
done < file.txt
</code>
Пример чтения из дескриптора:
<code bash>
exec 3<test.php # Открытие файла на чтение в дескрипторе 3.
read line <&3 # Чтение из дескриптора 3
exec 3>$-
echo $line
</code>
