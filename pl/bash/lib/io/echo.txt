<code bash>
echo строка
</code>
Утилита записывает переданные ей параметры в поток (по умолчанию используется стандартный поток вывода сценария).

Параметры:
  * ''-n'' - не добавлять символ новой строки в конец вывода;
  * ''-e'' - разрешить использовать спецсимволы.

Пример записи в поток вывода:
<code bash>
echo 'Hello world' # Вывод строки на экран.
</code>
Пример использования спецсимволов:
<code bash>
echo 'e' "1.\tHello world"
</code>
Пример записи в файл:
<code bash>
echo 'Hello world' >file.txt
</code>
Пример записи в дескриптор:
<code bash>
exec 3>file.txt
echo 'Hello world' >&3
exec 3>&-
</code>
