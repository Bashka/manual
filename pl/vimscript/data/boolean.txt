====== Литералы ======
В VimScript нет констант для обозначения логических значений ''true'' и ''false'', вместо этого приняты следующие соглашения:
  * К ''false'' относится число ноль. При этом строки для логической проверки так же приводятся к числовому типу;
  * Остальные, не составные типы данных являются ''true''.

====== Операторы ======
  * Операторы И, ИЛИ, НЕ:
    * ''выражение && выражение'' - оператор И;
    * ''выражение || выражение'' - оператор ИЛИ;
    * ''!выражение'' - оператор НЕ.
  * Математические:
    * ''число == число'' - равенство;
    * ''число != число'' - неравенство;
    * ''число > число'' - больше;
    * ''число >= число'' - больше или равно;
    * ''число < число'' - меньше;
    * ''%%число <= число%%'' - меньше или равно.
  * Строковые (любой из перечисленных операторов может быть дополнен символами ''?'' и ''#'' которые модифицируют оператор для сравнения без учета и с учетом регистра соответственно):
    * ''строка == строка'' - равенство;
    * ''строка =~ шаблон'' - соответствие регулярному выражению;
    * ''строка != строка'' - неравенство;
    * ''строка !~ шаблон'' - несоответствие регулярному выражению;
    * ''строка > строка'' - больше;
    * ''строка >= строка'' - больше или равно;
    * ''строка < строка'' - меньше;
    * ''%%число <= строка%%'' - меньше или равно.
  * Сравнение типа:
    * ''данные is данные'' - данные однотипны;
    * ''данные isnot данные'' - данные неоднотипны.

Пример сравнения строк с/без учета регистра:
<code vim>
echo 'hello' ==? 'Hello' " true
echo 'hello' ==# 'Hello' " false
</code>

====== Приведение ======
  * К числовому типу:
    * ''true'' -> ''1''
    * ''false'' -> ''0''
  * К строковому типу:
    * ''true'' -> ''%%'1'%%''
    * ''false'' -> ''%%'0'%%''
