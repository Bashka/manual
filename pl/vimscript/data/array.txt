====== Объявление и использование ======
Для инициализации массива используется запись:
<code vim>
let переменная = [значение, ...]
</code>
Здесь в качестве значения может выступать другой массив, что позволяет создавать вложенные массивы.

Для доступа к значению элемента массива используется запись (индексация начинается с нуля):
<code vim>
массив[индекс]
массив[индекс][индекс]... " Для многомерных массивов
</code>
При этом если в качестве индекса используется отрицательное значение, отсчет ведется с конца массива (индексация с -1).

Для установки значения элементу массива используется запись:
<code vim>
let массив[индекс] = значение
</code>

При передаче массива в переменную, он присваивается по ссылке:
<code vim>
let s:arr1 = [1, 2, 3]
let s:arr2 = s:arr1
let s:arr2[0] = 4
echo s:arr1 " [4, 2, 3]
</code>

====== Операции ======
Операции:
  * ''массив + массив'' - конкатенация массивов;
  * ''массив[начало:конец]'' - получение среза массива. Если в качестве конца диапазона используется отрицательная величина, то отсчет ведется с конца массива;
  * ''let [переменная, ...] = массив'' - раскрытие массива в переменные.
<code vim>
let s:arr = [1, 2, 3]
let s:arr[0:1] = [9, 8]
echo s:arr " [9, 8, 3]
</code>
