====== Запуск отладки ======
Запустить VimScript в режиме отладки можно одним из следующих способов:
  * ''vim -D файл'' - запуск файла с VimScript в режиме отладки;
  * '':debug команда'' - запуск команды в режиме отладки;
  * Использование точек останова.

===== Точки останова =====
Точки останова определяют строки, при достижении которых интерпретатор VimScript перейдет в режим отладки.

Для установки точки останова на указанной позиции в строке используется команда ''breakadd file номерСтроки файл''.
<code vim>
breakadd file 3 ./test.vim
</code>

Для удаления точки останова используется команда ''breakdel номерТочки''. Если в качестве номера точки указать ''*'', то будут удалены все точки останова.

Номер удаляемой точки останова можно получить из списка установленных точек останова с помощью команды ''breaklist''.

====== Режим отладки ======
В режиме отладки строки сценария интерпретируются в интерактивном режиме. При этом пользователь может вводить любые команды, на пример для получения содержимого переменной или для вызова функций.

В этом режиме так же доступны следующие команды:
  * ''c'' - продолжить выполнение до следующей точки останова;
  * ''q'' - закончить выполнение сценария;
  * ''n'' - выполнить текущую команду и вернуться в режим отладки;
  * ''s'' - выполнить текущую команду, перейдя в вызываемую функцию или подключаемый файл;
  * ''f'' - продолжить выполнение до выхода из функции или подключаемого файла.
