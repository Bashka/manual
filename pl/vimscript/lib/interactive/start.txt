  * ''confirm(сообщение, варианты, [поУмолчанию])'' - предоставить пользователю возможность выбора одного из вариантов. Варианты задаются в виде строки, разделенной символами новой строки с использованием символа ''&'' перед буквой варианта;
<code vim>
let s:choice = confirm("What do you want?", "&Apples\n&Oranges\n&Bananas", 2)
</code>
  * ''getcharmod()'' - получение модификатора, использованного при вводе символа пользователем для функции ''getchar'';
  * ''getchar()'' - запрос символа у пользователя;
  * ''inputsecret(сообщение, [значение])'' - аналогично функции ''input'', но ввод пользователя не будет отображаться на экране;
  * ''input(сообщение, [значение])'' - запрос строки у пользователя. Если второй параметр задан, его значение будет использовано в качестве значения ввода для диалога;
<code vim>
let s:color = input('Color?', 'white')
</code>
