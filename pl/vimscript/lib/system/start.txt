  * ''delete(файл)'' - удалить файл;
  * ''executable(файл)'' - проверка возможности запуска указанного файла или утилиты;
  * ''filereadable(файл)'' - проверка возможности прочитать из файла;
  * ''filewritable(файл)'' - проверка возможности записать в файл;
  * ''fnamemodify(файл, флаги)'' - преобразовать имя файла. В качестве флагов могут выступать следующие: '':p'' - полный путь к файлу; '':~'' - путь к файлу от домашнего каталога; '':.'' - путь к файлу относительно текущего каталога; '':h'' - адрес каталога, в котором хранится файл; '':t'' - имя и тип файла; '':e'' - тип файла; '':r'' - адрес файла без расширения (может применяться несколько раз); ''gs?шаблон?замена?'' - замена всех вхождений шаблона на строку; ''s?шаблон?замена?'' - замена первого вхождения шаблона на строку;
  * ''getcwd()'' - функция возвращает адрес текущего каталога (в котором был запущен Vim);
  * ''getfsize(файл)'' - получить размер файла;
  * ''getftime(файл)'' - получить дату последнего изменения файла в виде временной метки;
  * ''globpath(каталог, шаблон)'' - поиск файлов по заданному шаблону (glob) в указанном каталоге (может быть перечисленно несколько каталогов через запятую);
  * ''hostname()'' - получить имя системы;
  * ''isdirectory(каталог)'' - проверка существования указанного каталога;
  * ''localtime()'' - получить текущее время в виде временной метки;
  * ''mkdir(каталог)'' - создать каталог;
  * ''readfile(файл)'' - функция читает из файла и возвращает его содержимое;
  * ''rename(файл, новыйАдрес)'' - переименовать/переместить файл;
  * ''strftime(формат, [временнаяМетка])'' - преобразовать временную метку в строку даты с заданным форматом. В параметре формата могут использоваться следующие метки для обозначения времени: ''%Y'' - год в формате ГГГГ; ''%y'' - год в формате ГГ; ''%d'' - день; ''%m'' - месяц; ''%b'' - трехбуквенное название месяца; ''%B'' - полное название месяца; ''%H'' - часы; ''%M'' - минуты; ''%S'' - секунды; ''%T'' - время в формате ЧЧ:ММ:СС; 
  * ''system(команда)'' - выполнить команду в оболочке и получить результат в виде строки;
  * ''tempname()'' - получить адрес временного файла;
  * ''writefile(строки, файл)'' - перезапись файла массивом строк.
