  * ''bufexists(буфер)'' - функция проверяет, существует ли буфер. В качестве параметра можно передать как номер целевого буфера, так и его имя (адрес файла);
  * ''bufloaded(буфер)'' - функция проверяет, загружен ли буфер. В качестве параметра можно передать как номер целевого буфера, так и его имя (адрес файла);
  * ''bufname(номерБуфера)'' - получение имени буфера (адрес файла) по его номеру;
  * ''bufnr(буфер, [создать])'' - получение номера буфера. В качестве первого параметра можно использовать следующие выражения: ''$'' - последний буфер; ''%'' - текущий буфер. Если буфер не существует, функция возвращает -1. Если в качестве второго параметра передано ''1'', то буфер будет создан в случае его отсутствия, а функция вернет номер нового буфера;
<code vim>
let s:newBuffer = bufnr('file.txt', 1)
</code>
  * ''bufwinnr(номерБуфера)'' - определение окна, в котором загружен целевой буфер. Если буфер не загружен в окне, функция возвращает -1;
  * ''getbufvar(номерБуфера, переменная)'' - получения зачения переменной целевого буфера;
  * ''setbufvar(номерБуфера, переменная, значение)'' - установка значения переменной для целевого буфера.
