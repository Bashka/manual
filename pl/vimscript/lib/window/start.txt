  * ''getwinvar(номерОкна, переменная)'' - получить значение переменной окна;
  * ''setwinvar(номерОкна, переменная, значение)'' - установить значение переменной окна;
  * ''winbufnr(номерОкна)'' - функция определяет номер буфера, загруженного в целевое окно;
  * ''winheight(номерОкна)'' - для горизонтально-ориентированных окон определяет высоту активного окна;
  * ''winnr([окно])'' - функция определяет номер окна. В качестве первого параметра можно использовать одно из следующих значений: ''$'' - последнее окно; ''#'' - предыдущее активное окно. Если параметр не передан, возвращается номер активного окна;
  * ''winwidth(номерОкна)'' - для вертикально-ориентированных окон определяет ширину активного окна.
