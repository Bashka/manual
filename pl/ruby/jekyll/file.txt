====== Статичные файлы ======

Любые файлы, не начинающиеся с вводной, будут скопированы при компиляции.

====== Шаблонизируемые файлы ======

Файлы, начинающиеся с вводной и допускающие обработку, будут скомпилированны и скопированы. К таким файлам относятся:

  * HTML (''*.html'') - файлы с HTML разметкой будут скомпилированы в HTML
  * Markdown (''*.md'', ''*.markdown'') - файлы с Markdown разметкой будут скомпилированы в HTML
  * SASS (''*.scss'') - файлы с SASS будут скомпилированы в CSS
  * CoffeeScript (''*.coffee'') - файлы с CoffeeScript будут скомпилированы в JS

Файлы с разметкой, в которых указана переменная ''page.permalink'' использующая "Человеко-читаемый URL" (на пример ''/about/'') будут скомпилированы в файл ''index.html'' указанного ссылкой каталога (на пример в ''/about/index.html''):

<code html>
---
permalink: /contacts/
---
<div>
  ...
</div>
</code>

====== Шаблоны ======

Шаблоны (Layouts) используются для встраивания в них контента конкретной страницы. По умолчанию файлы шаблонов располагаются в каталоге ''_layouts'' и применяются к странице в переменной ''page.layout''.

Файл ''_layouts/default.html''
<code html>
<html>
  <head>...</head>
  <body>
    ...
    {{ content }}
    ...
  </body>
</html>
</code>

Файл ''page.html'':
<code html>
---
layout: default
---
<div>
  ...
</div>
</code>

====== Блоки ======

Файлы, расположенные в каталоге ''_includes'' могут быть подключены к странице через команду ''%%{% include файл %}%%''.

Подключение файла ''_includes/footer.html'':

<code html>
{% include  footer.html %}
</code>

Для передачи параметров фрагменту, их можно перечислить в виде атрибутов команды ''include''. Во фрагменте они будут доступны в переменной ''include.параметр'':

<code html>
{% include  footer.html paramA="foo" %}
</code>
