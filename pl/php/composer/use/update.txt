===== Обновление =====

Для обновления зависимостей до более новых версий (переустановки зависимостей) используется следующая команда:

<code bash>
php composer.phar update
</code>

Данная команда ориентируется на файл ''composer.json'', а не на ''composer.lock'' и обновляет последний. Если необходимо обновить сам файл блокировки ''composer.lock'', используется команда:

<code bash>
php composer.phar update --lock
</code>

===== Удаление =====

Для удаления зависимостей необходимо:

  * Удалить зависимость из файла ''composer.json'' самостоятельно или с помощью команды:

<code bash>
php composer.json remove издатель/пакет
</code>

  * Выполнить обновление зависимостей проекта, как описано выше.
