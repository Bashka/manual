====== Семантика ======

===== EventInterface =====

В качестве объектных представлений выбрасываемых событий должен выступать класс, реализующий интерфейс ''EventInterface'', включающий следующие основные методы:

  * ''setName/getName'' - устанавливает и получает имя события
  * ''setTarget/getTarget'' - устанавливает и получает объект, сгенерировавший событие
  * ''setParam/getParam'' - устанавливает и получает дополнительный параметр события
  * ''setParams/getParams'' - устанавливает и получает дополнительные параметры события
  * ''stopPropagation'' - останавливает дальнейшую обработку (всплытие) события
  * ''propagationIsStopped'' - определяет, остановлена ли дальнейшая обработка события

====== Реализации ======

===== Event =====

Класс ''Event'' является реализацией интерфейса ''EventInterface'' в ZF.

<code php>
$em = new EventManager;
$em->attach('MyEvent', ...);
$em->trigger(new Event('MyEvent', $target, $params));
</code>
