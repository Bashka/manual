====== Семантика ======

===== ConfigInterface =====

Для конфигурирования механизмов сессии в ZF используется специальный класс-конфигурация, реализующий интерфейс ''ConfigInterface''.

Методы:

  * ''setOptions/getOptions'' - устанавливает опции конфигурации и предоставляет значения не специальных опций
  * ''toArray'' - предоставляет значения всех опций конфигурации в виде массива
  * ''setOption/getOption/hasOption'' - устанавливает и получает значение опции конфигурации, а так же проверят наличие данной опции

Опции:

  * ''save_path'' - адрес, передаваемый в [[.:save_handler|SaveHandler]] и определяющий место для сохранения данных сессии в постоянном хранилище
  * ''name'' - имя сессии
  * ''cookie_lifetime'' - время хранения cookie в секундах

====== Реализация ======

===== StandardConfig =====

Класс ''StandardConfig'' реализует интерфейс ''ConfigInterface'' и включает базовые опции для конфигурирования механизма сессий.

===== SessionConfig =====

Класс ''SessionConfig'' расширяет ''StandardConfig'' дополняя его опциями, позволяющими сконфигурировать опции PHP интерпретатора (INI), связанные с сессией.
