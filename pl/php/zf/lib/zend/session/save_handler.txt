====== Семантика ======

===== SaveHandlerInterface =====

Для сохранения полезных данных сессии в постоянное хранилище (через глобальный массив ''$_SESSION'') должен использовать класс, реализующий интерфейс ''SaveHandlerInterface'', который, в свою очередь, расширяет стандартный интерфейс PHP ''SessionHandlerInterface''.

  * ''open/close'' - открывает и закрывает соединение с постоянным хранилищем
  * ''read/write'' - считывает и записывает данные в постоянное хранилище под ключом идентификатора сессии
  * ''destroy'' - уничтожает данные в постоянном хранилище, хранящиеся под данным идентификатором сессии
  * ''gc'' - удаляет "старые" сессии
