Фабрика [[http://framework.zend.com/manual/current/en/modules/zend.loader.autoloader-factory.html|AutoloaderFactory]] позволяет использовать несколько загрузчиков классов одновременно используя единую конфигурацию. Такой подход позволяет ускорить автозагрузку классов благодаря смешению различных подходов и выбору наиболее подходящего решения для конкретного класса (на пример частная загрузка с помощью ClassMapAutoloader с дозагрузкой не определенных в карте классов с помощью StandardAutoloader).

После подключения данной фабрики необходимо ее сконфигурировать. Делается это следующим образом:

<code php>
require_once 'Zend/Loader/AutoloaderFactory.php';
Zend\Loader\AutoloaderFactory::factory([
  'Zend\Loader\ClassMapAutoloader' => [
    __DIR__ . '/autoload_classmap.php'
  ],
  'Zend\Loader\StandardAutoloader' => [
    'autoregister_zf' => true,
    'namespaces' => [
        'MyLib' => __DIR__ . '/MyLib',
    ]
  ]
]);
</code>
