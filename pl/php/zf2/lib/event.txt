Функция-обработчик события, закрепляемая с помощью метода ''attach'' класса [[ЯП:ZF2:lib:EventManager]] получает в качестве параметра экземпляр класса ''Event''. Этот класс имеет следующие основные методы:

  * getName - метод возвращает имя события;
  * getTarget - метод возвращает объект, инициировавший событие;
  * getParams - метод возвращает параметры события;
  * stopPropagation - метод прекращает вызов остальных обработчиков данного события.
