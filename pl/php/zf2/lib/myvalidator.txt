[[:start|Главная]] > [[:ЯП:ZF2|Zend Framework 2]] > Компоненты фреймворка > **Собственные валидаторы**

Для создания собственного класса для валидации данных, достаточно реализовать интерфейс ''ValidatorInterface''.

<code php>
class MyNewValidator implements Zend\Validator\ValidatorInterface{
  public function isValid($value){
    if(...){
      return true;
    }
    else{
      return false;
    }
  }
  
  public function getMessage(){
    return ...; // Массив сообщений, объясняющих причину провала валидации
  }
}
</code>

Зарегистрировать собственный валидатор, чтобы к нему можно было обращаться по имени, можно следующим образом: 

<code php>
StaticValidator::getPluginManager()->setInvokableClass(
  'myNewValidator', 'MyNewValidator'
);
echo StaticValidator::isValid('Data', 'myNewValidator');
</code>
