[[:start|Главная]] > [[:ЯП:ZF2|Zend Framework 2]] > Компоненты фреймворка > **Собственные фильтры**

Для создания собственного класса для фильтрации данных, достаточно реализовать интерфейс [[http://framework.zend.com/manual/current/en/modules/zend.filter.writing-filters.html|FilterInterface]].

<code php>
class MyFilter implements Zend\Filter\FilterInterface{
  public function filter($value){
    ...
    return $valueFiltered;
  }
}
</code>

Зарегистрировать собственный фильтр, чтобы к нему можно было обращаться по имени, можно следующим образом:

<code php>
StaticFilter::getPluginManager()->setInvokableClass(
  'myNewFilter', 'MyNewFilter'
);
echo StaticFilter::execute('Data', 'myNewFilter');
</code>
