Обычно, файловая структура проекта, реализуемого с использованием MVC ZF2, имеет следующий вид:
    project
      config - конфигурации проекта
      data - кэш и временные файлы проекта
      module - модули проекта
      public - клиенские файлы проекта и index.php
      vendor - библиотеки проекта
Данная структура реализуется при сборке проекта с использованием [[https://github.com/zendframework/ZendSkeletonApplication|ZendSkeletonApplication]].
