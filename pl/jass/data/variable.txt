====== Объявление и использование ======
Переменные в Jass деляться на два класса, в зависимости от области видимости:
  * Глобальные - переменные объявляются в блоке ''globals'' и видны всем частям скрипта;
<code>
globals
string myStr = "Hello world" // Глобальная переменная
endglobals
</code>
  * Локальные - переменные объявляются с помощью ключевого слова ''local'' в теле функции и видны только внутри него.
<code>
local string myStr = "Hello world" // Локальная переменная
</code>

При объявлении переменной необходимо указать ее тип:
<code>
тип имя = значение
</code>

Для присвоения значения переменной используется запись:
<code>
set переменая = значение
</code>

Для получения значения переменной достаточно указать ее имя в месте использования:
<code>
local string obj = "world"
local string str = "Hello " + obj
</code>

====== Операции ======
  * ''constant объявление переменной'' - сделать глобальную переменную константной;
<code>
globals
constant int PI = 3.14
endglobals
</code>
