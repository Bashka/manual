====== Объявление и использование ======
Для объявления массива используется запись:
<code>
модификаторы тип array имя
</code>
Созданный массив инициализируется нулевыми значениями.

Для доступа к значению элемента массива используется запись (индексация начинается с нуля):
<code>
массив[индекс]
</code>

Для установки значения элементу массива используется запись:
<code>
set имя[индекс] = значение
</code>
Важно помнить, что один массив нельзя присвоить другому.

Максимальная длина массива указана в константе ''JASS_MAX_ARRAY_SIZE''.
