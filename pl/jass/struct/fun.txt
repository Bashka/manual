====== Объявление ======
Для объявления функции используется запись:
<code>
function имя takes параметры returns возврат
  тело
endfuncton
</code>

Пример объявления функции:
<code>
function myFun takes player p, string message returns nothing
    call DisplayTextToPlayer(p, 0.0, 0.0, message)
endfunction
</code>
Эта функция принимает два параметра и не возвращает ничего.

====== Вызов ======
Для вызова функции используется команад ''call'':
<code>
call myFun(Player(0), "Hello world")
</code>

Если функция возвращает некоторое значение, ее можно вызвать внутри выражения или другой команды:
<code>
local player p = Player(0)
</code>

====== Возвращаемое значение ======
Для возврата значения из функции используется команда ''return значение''. После возврата значения, функция прекращает работу.

Важно, чтобы возвращаемое значение было того же типа, что объявлено в объявлении функции.

====== Косвенные ссылки ======
На функцию можно ссылаться из переменной. Для этого используется команда ''function имя'', которая возвращает ссылку на функцию:
<code>
local code newFun = function myFun
</code>
