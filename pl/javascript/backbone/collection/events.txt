====== События коллекции ======
Коллекция может генерировать следующие события:

  * ''add'' - выбрасывается при добавлении в коллекцию экземпляра модели
  * ''remove'' - выбрасывается при удалении из коллекции экземпляра модели
  * ''reset'' - выбрасывается при установке значений коллекции
  * ''change'' - выбрасывается при изменении состояния любого экземпляра модели, входящего в коллекцию
  * ''sort'' - выбрасывается при сортировке коллекции
  * ''all'' - выбрасывается при возникновении любого из перечисленных выше событий
