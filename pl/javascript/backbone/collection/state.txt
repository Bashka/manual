====== Состояние коллекции ======
Состояние коллекции хранится в виде массива экземпляров модели в свойстве ''models'' экземпляра. Доступ к элементам этого массива можно получить с помощью следующих методов:

  * ''at(id)'' - возвращает модель по ее идентификатору или индексу массива
  * ''get(id)'' - возвращает модель по ее идентификатору или ''cid''
  * ''add(модель)'' - добавляет модель в коллекцию и вызывает метод ''sort''
  * ''remove(модель)'' - удаляет модель из коллекции
  * ''reset(массив)'' - заменяет содержимое коллекции указанным массивом моделей
  * ''sort()'' - выполняет сортировку модели с помощью функции-компаратора

Экземпляр коллекции так же включает следующие свойства:

  * ''length'' - число элементов, добавленных в коллекцию
