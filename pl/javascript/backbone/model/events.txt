====== События модели ======
Модель может генерировать следующие события:

  * ''change'' - выбрасывается в случае изменения состояния экземпляра модели
  * ''change:свойство'' - выбрасывается в случае изменения значения конкретного свойства экземпляра модели
  * ''invalid'' - выбрасывается при ошибки валидации методом ''validate()''
  * ''destroy'' - выбрасывается при успешном выполнении метода ''destroy()''
  * ''sync'' - выбрасывается в случае успешной синхронизации экземпляра модели с сервером
  * ''error'' - выбрасывается в случае ошибки при синхронизации экземпляра модели с сервером
  * ''all'' - выбрасывается при возникновении любого из перечисленных выше событий
