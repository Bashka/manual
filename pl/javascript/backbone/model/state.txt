====== Состояние модели ======
Состояние модели хранится в виде словаря в свойстве ''attributes'' экземпляра. Доступ к значениям свойств, хранящихся в этом словаре, можно получить с помощью следующих методов:

  * ''has(свойство)'' - проверяет, установлено ли данное свойство у экземпляра модели
  * ''get(свойство)'' - получает значение свойства экземпляра модели
  * ''escape(свойство)'' - получает XSS-безопасное значение свойства экземпляра модели
  * ''set(свойство, значение, [опции])'' - устанавливает значение свойству экземпляра модели. В качестве опций, могут выступать следующие значения:
    * ''validate'' - если передано ''true'', перед установкой значения будет выполнена валидации. Ошибки при валидации не позволят изменить состояние экземпляра сущности и приведут к генерации исключения ''invalid''
  * ''unset(свойство)'' - удаляет свойство экземпляра модели
  * ''clear()'' - удаляет все свойства экземпляра модели
  * ''isValid()'' - выполняет валидацию состояния модели с помощью метода ''validate()''

Экземпляр модели так же включает следующие свойства:

  * ''id'' - идентификатор экземпляра на стороне сервера. Данное свойство устанавливается только после синхронизации экземпляра модели с сервером
  * ''cid'' - идентификатор экземпляра на стороне клиента. Данное свойство устанавливается автоматически при создании экземпляра модели
